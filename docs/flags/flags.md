# Flags Used in Cube3D

## Linker/Compiler Flags

- `-Lminilibx-linux`: Search for libraries in minilibx-linux directory.
- `-lmlx`: Link MiniLibX library.
- `-lX11`: Link X11 graphics library.
- `-lXext`: Link X11 extensions library.

These flags are required to build and run Cube3D on Linux with MiniLibX.
