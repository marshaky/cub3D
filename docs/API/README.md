# cub3d API Reference

This section documents internal APIs used by the cub3d engine.

Each API has its own documentation file describing purpose,
parameters and internal logic.

Writing standard:

- [API template](TEMPLATE.md)

---

## API list

| API name | Description |
|--------|--------|
| [app](app/README.md) | Application and camera initialization. |
| [hooks](hooks/README.md) | Input and window event handlers. |
| [map](map/README.md) | Map queries and map bootstrap helpers. |
| [raycast](raycast/README.md) | DDA ray traversal setup and casting. |
| [loader](loader/README.md) | Texture loading API. |
| [player](player/README.md) | Movement, rotation and per-frame input update. |
| [render](render/README.md) | Frame rendering and wall-column drawing. |
| [minimap](minimap/README.md) | Minimap rendering helpers and overlays. |
| [utils](utils/README.md) | Shutdown and utility helpers. |
